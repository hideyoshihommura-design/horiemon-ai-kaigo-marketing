## 1. ペライチの基本操作

### 1-1. ログイン

1. ブラウザで **https://peraichi.com** にアクセス
2. 右上の「ログイン」をクリック
3. メールアドレスとパスワードを入力してログイン

### 1-2. 編集画面の開き方

1. ログイン後、「マイページ」に移動する
2. 編集したいページ（介護校のLP）の「編集」ボタンをクリック
3. 編集画面が開く

### 1-3. ペライチの編集画面の見方

```
┌─────────────────────────────────┐
│  上部メニューバー                    │
│  [ブロック挿入] [ページ設定] [プレビュー] [公開]  │
├─────────────────────────────────┤
│                                 │
│  ┌───────────────────────┐      │
│  │ セクション（ブロック）1    │      │
│  │ ※カーソルを合わせると     │      │
│  │  編集ボタンが表示される   │      │
│  └───────────────────────┘      │
│         ＋ ← ブロック追加ボタン       │
│  ┌───────────────────────┐      │
│  │ セクション（ブロック）2    │      │
│  └───────────────────────┘      │
│         ＋                       │
│  ┌───────────────────────┐      │
│  │ セクション（ブロック）3    │      │
│  └───────────────────────┘      │
│                                 │
└─────────────────────────────────┘
```

### 1-4. 基本操作一覧

| やりたいこと | 操作方法 |
|-----------|--------|
| テキストを変更する | テキスト部分をクリック → 直接編集 |
| 画像を変更する | 画像をクリック → 「画像を変更」→ アップロードまたは選択 |
| ボタンのリンクを変更する | ボタンをクリック → 「リンク先URL」を入力 |
| ボタンの色を変更する | ボタンをクリック → 「カラー設定」から選択 |
| 新しいセクションを追加する | セクション間の「＋」ボタンをクリック → テンプレートから選択 |
| セクションの順番を変える | セクション左端の「↑↓」矢印をクリック |
| セクションを削除する | セクション右上の「×」ボタンをクリック |
| セクションの背景色を変える | セクションの「編集」→「背景設定」 |
| フォームを設置する | 「＋」→「フォーム」カテゴリからテンプレートを選択 |
| HTMLを埋め込む | 「＋」→「HTML埋め込み」ブロックを選択 |

### 1-5. 編集時の注意点

- **こまめに保存する**: 右上の「保存」ボタンを定期的にクリック
- **プレビューで確認**: 変更後は「プレビュー」で実際の表示を確認する
- **スマホ表示も確認**: プレビュー画面でスマホ表示に切り替えて確認する（介護現場はスマホ閲覧が多い）
- **公開は最後に**: すべての編集が完了してから「公開」ボタンを押す

### セクション1: ファーストビュー（最重要）

**目的**: 介護事業所の施設長・管理者が一目で「自分向けだ」とわかること

#### 操作手順

1. 現在のファーストビューのセクションをクリック
2. テキストを以下に変更する

#### 変更する内容

**キャッチコピー（大見出し）:**
```
介護事業所のAI人材育成
助成金で実質 月5,833円/人〜
```

**サブコピー（小見出し）:**
```
堀江貴文プロデュース｜法人向け生成AI研修 No.1（4冠）
240以上の動画講義＋チャットサポート＋月1回の伴走MTG
介護・在宅医療の現場で即使えるAI活用法を学べます
```

**CTAボタンを2つ設置:**

| ボタン | テキスト | 色 | リンク先 |
|-------|---------|---|---------|
| 主CTA | 無料ウェビナーに申し込む | 赤またはオレンジ（目立つ色） | ウェビナー申込みセクションへのページ内リンク（#webinar） |
| 副CTA | 資料を無料ダウンロード | 白枠（アウトライン） | 資料請求セクションへのページ内リンク（#download） |

#### CTAボタンの設置方法

1. ファーストビューのセクション内で「ボタン」をクリック
2. ボタンのテキストを上記に変更
3. 「リンク設定」で「ページ内リンク」を選択
4. リンク先を対応するセクションのアンカーに設定
5. ボタンがない場合は、セクション内の「＋（要素追加）」からボタンを追加

#### ページ内リンクの設定方法

1. リンク先にしたいセクション（例: ウェビナー申込みセクション）を選択
2. セクションの「編集」→「セクションID」に `webinar` と入力
3. ボタンのリンク先に `#webinar` と入力
4. 同様に資料ダウンロードセクションのIDには `download` と設定

#### 背景画像について

- 現在の堀江貴文の写真入りの画像は維持してOK
- ただし、テキストが読みにくい場合は背景を暗くするオーバーレイを設定する
  - セクション編集 → 背景設定 → 「オーバーレイ」→ 黒を選択 → 不透明度30-40%

**目的**: 信頼性を一瞬で伝える

1. ファーストビューの下の「＋」をクリック

- 満足度 No.1

```

※「法人向け生成AI研修サービス」に関する市場調査《No.1検証調査》
```

```
法人向け生成AI研修 No.1（4冠）
実績社数伸び率 No.1 ／ 講義の分かりやすさ No.1 ／ 信頼度 No.1 ／ 満足度 No.1
```

### セクション10: ウェビナー申込みセクション（新規追加・主CTA）

**目的**: LP全体で最も重要なコンバージョンポイント

#### 操作手順

1. 「＋」→「フォーム」カテゴリからフォーム付きのテンプレートを選択
2. セクションIDに `webinar` を設定（ページ内リンク用）

#### セクションIDの設定方法

1. このセクションの編集画面を開く
2. 「セクション設定」や「アンカーリンク設定」を探す
3. ID欄に `webinar` と入力

#### 掲載する内容

**見出し:**
```
無料ウェビナー開催中
「介護現場のAI活用最前線 - 記録業務を半分にする方法」
```

**ウェビナー情報:**
```
日時: 毎月第2・第4木曜日 14:00〜15:00
形式: Zoom Webinar（オンライン・参加無料）
内容: 介護現場でのAI活用デモ、助成金の活用方法、質疑応答
```

**フォーム項目（3項目以内に絞る）:**

| 項目名 | 入力形式 | 必須/任意 |
|-------|---------|---------|
| お名前 | テキスト | 必須 |
| メールアドレス | メール | 必須 |
| 法人名・施設名 | テキスト | 任意 |

#### フォームの設定方法

1. フォームブロックの「編集」をクリック
2. 「項目を追加」で上記3項目を設定
3. 「送信先メールアドレス」に自社のメールアドレスを入力
4. 「送信ボタンのテキスト」を「ウェビナーに申し込む（無料）」に変更
5. 「送信完了メッセージ」を設定:
   ```
   お申し込みありがとうございます。
   ご登録いただいたメールアドレスにウェビナーの詳細をお送りしました。
   ```

#### 背景色

- 背景色: 薄い青（#e8f4fd）またはペライチの「ライトブルー」
- 他のセクションと差をつけて目立たせる

### 4-3. コンバージョン計測の設定

ペライチのフォーム送信をコンバージョンとして計測するために、フォームの「送信完了ページ」にイベントコードを設置する。

#### ペライチのフォーム設定で

1. フォームの「編集」→「送信完了後の動作」

## 補足: ペライチのプランについて

ペライチには複数のプランがあり、プランによって使える機能が異なります。
この改修に必要な機能は以下の通りです:

| 機能 | 必要なプラン |
|------|-----------|
| 基本的なテキスト・画像編集 | 全プラン |
| フォーム設置 | ライトプラン以上 |
| HTMLタグ埋め込み（GA4・ピクセル） | レギュラープラン以上 |
| 独自ドメイン | レギュラープラン以上 |
| A/Bテスト | ビジネスプラン |

現在のプランでフォームやHTML埋め込みが使えない場合は、プランのアップグレードが必要です。
