```

  └→ 介護校の取り分: 50% = 15.5万円/人
  └→ 本部の取り分:   50% = 15.5万円/人
```

- **1契約あたりの平均受講人数を4名と仮定**した場合:

### リード→商談の転換率の根拠

リードの種類によって商談化率が異なる。

### 流入→リード転換の分解分析（LP CVRモデル）

上記のファネルでは「リード数」を起点にしているが、リードを獲得するためにはまずLPへのトラフィック（流入）が必要である。ここでは広告予算がLP訪問数→リード数にどう変換されるかを分解する。

#### 基本モデル

```
広告予算 ÷ CPC（クリック単価） = LP訪問数
LP訪問数 × CVR（転換率） = リード数（ウェビナー申込み or 資料請求）
```

#### チャネル別CPC・CVR想定

| チャネル | CPC想定 | LP CVR想定 | 根拠 |
|---------|---------|-----------|------|
| Facebook広告 | 300-600円 | 3-5% | 介護業界は競合が少ないためCPCは比較的低い。ただしターゲットが狭いため配信量に限界あり |
| Google検索広告 | 500-800円 | 4-7% | 検索意図が高いためCVRは良好。ただし「介護 AI研修」の月間検索ボリュームが限られる |
| LINE広告 | 200-400円 | 2-3% | CPCは安いがB2Bの意図が低く、CVRは控えめ |
| LinkedIn広告 | 800-1,500円 | 3-5% | CPCは高いがターゲット精度が高くCVRも良い |

#### プランB（月85万円）の流入→リード分解

**悲観シナリオ（CPC上振れ × CVR下振れ）**

| チャネル | 予算 | CPC | 訪問数 | CVR | リード数 |
|---------|------|-----|--------|-----|---------|
| Facebook広告 | 35万円 | 600円 | 583 | 3% | 17件 |
| Google広告 | 18万円 | 800円 | 225 | 4% | 9件 |
| LINE広告 | 4万円 | 400円 | 100 | 2% | 2件 |
| LinkedIn広告 | 5万円 | 1,500円 | 33 | 3% | 1件 |
| DM・オーガニック | - | - | - | - | 7件 |
| **合計** | | | **941** | | **36件** |

**楽観シナリオ（CPC良好 × CVR上振れ）**

| チャネル | 予算 | CPC | 訪問数 | CVR | リード数 |
|---------|------|-----|--------|-----|---------|
| Facebook広告 | 35万円 | 300円 | 1,167 | 5% | 58件 |
| Google広告 | 18万円 | 500円 | 360 | 7% | 25件 |
| LINE広告 | 4万円 | 200円 | 200 | 3% | 6件 |
| LinkedIn広告 | 5万円 | 800円 | 63 | 5% | 3件 |
| DM・オーガニック | - | - | - | - | 15件 |
| **合計** | | | **1,790** | | **107件** |

#### 分析からの示唆

- **悲観シナリオでは36件/月** — チャネル別CPA前提の下限値（50件）を下回る可能性がある。これはCPCとCVRの両方が悪化した最悪ケースであり、実際にはこのレベルが持続することは考えにくいが、立ち上げ初月に一時的にこの水準になるリスクはある
- **LP CVRが最も重要なレバー。** CPCは市場相場に大きく左右されるが、LP CVRは自社の改善努力で引き上げられる。CVRが3%→5%に改善するだけでリード数は1.67倍になる
- **LP CVRの目標値: 4%以上を安定的に維持する** — これを下回る場合はLP改善を最優先で実施する

#### LP CVR改善のための条件

| 施策 | CVRへの影響 | 実施難度 |
|------|-----------|---------|
| ファーストビューの介護特化メッセージ | 高（離脱率の低下） | 低 |
| ウェビナー申込みフォームの簡略化（3項目以内） | 高（フォーム完了率の向上） | 低 |
| 助成金シミュレーターの設置 | 中（滞在時間の延長・CV追加） | 中 |
| 導入事例の追加（社名入り） | 中（信頼性の向上） | 中（事例取得が必要） |
| モバイル最適化（介護現場はスマホ利用が多い） | 高（モバイルCVRの底上げ） | 低 |
| ページ読み込み速度の改善（3秒以内） | 中（離脱防止） | 低〜中 |

**LP CVRの計測と改善はフェーズ1（基盤構築）で着手し、フェーズ3（最適化期）でヒートマップ分析を含めた本格改善を行う。** 広告費を投入する前にLP CVRが最低3%を確保できていることを確認すべきである。

### 予算と流入数の整合性評価

#### 結論: 現実シナリオでは成立するが、マージンは薄い

プランB（月85万円）のデジタル広告予算は実質62万円（Facebook 35万 + Google 18万 + LINE 4万 + LinkedIn 5万）。この予算で必要リード数を確保できるかを検証する。

| | 必要リード数 | 悲観シナリオ | 現実シナリオ | 楽観シナリオ |
|---|---|---|---|---|
| 初期（1-3ヶ月） | 50-73件 | **36件（不足）** | 60-65件（成立） | 107件（余裕） |
| 安定期（4ヶ月〜） | 65-95件 | 47件（不足） | 78-85件（成立） | 140件（余裕） |

**悲観シナリオ（LP CVR 3%以下 × CPC上振れ）では必要リード数に対し30-50%不足する。**

#### 予算が成立する3つの条件

以下を**同時に**満たす必要がある:

| 条件 | 基準値 | 達成確度 |
|------|--------|---------|
| LP CVRが4%以上 | 改善前は未達の可能性が高い。介護法人向けに最適化後であれば達成可能 | **LP改善が前提** |
| Facebook広告のCPCが500円以下 | 介護業界は広告競合が少なく可能性は高いが、配信開始まで不明 | 中〜高 |
| 広告外リード（DM・オーガニック・紹介）が月7件以上 | SEO・紹介チャネルが育つまで3-6ヶ月かかる | 初期は低い |

#### 対策: 段階的な予算投入（推奨）

悲観シナリオのリスクを回避するため、**75万円を初月から投入せず、段階的に引き上げる**運用を推奨する。

```
【1ヶ月目】検証フェーズ: 月30万円
  ├─ LP改修を自社で最優先で完了させる（費用ゼロ）
  ├─ Facebook・Google広告を少額（各10-15万円）でテスト出稿
  ├─ CPC・CVRの実数値を取得
  └─ 目的: 75万円投入の判断材料を得る

【2ヶ月目】判断フェーズ
  ├─ LP CVRが4%以上 → 75万円に引き上げ
  ├─ LP CVRが3-4% → LP追加改善しつつ50万円で継続
  └─ LP CVRが3%未満 → 広告は最小限に抑え、LP改善に集中

【3ヶ月目以降】本格投入フェーズ
  └─ CPC・CVRの実績データに基づき75万円を配分
```

**この運用により、初月の30万円の範囲で「75万円投入しても大丈夫か」を検証できる。** 検証なしに75万円を投入し、LP CVRが2%だった場合、月20-30万円の広告費が無駄になる。初月の慎重な投資で、その後の6ヶ月分の投資判断の精度が上がる。

#### 初月の予算配分案（検証フェーズ: 月30万円）

| 項目 | 月額 | 目的 |
|------|------|------|
| Facebook広告（テスト） | 12万円 | CPC・CVRの実数値取得 |
| Google広告（テスト） | 8万円 | 検索広告のCPC・CVR取得 |
| スタジオレンタル | 2万円 | 第1回ウェビナー撮影 |
| メール配信ツール | 1万円 | シーケンス構築 |
| DM制作・リスト準備 | 7万円 | 2ヶ月目の送付に向けた準備 |
| **合計** | **約30万円** | |

※LP改修・コンテンツ制作は自社対応（費用ゼロ）

**初月は広告費20万円でテスト出稿し、LP CVR・CPC・リードの質を検証する。** この20万円で得られるデータ（LP訪問数500-1,000件程度）があれば、CVRの実力値が見える。

### リード数を底上げする施策（広告費に依存しない）

広告のCPAが上振れした場合に備え、以下の施策でリード不足を補う。

| 施策 | 効果 | コスト |
|------|------|--------|
| ウェビナー参加者への3日以内の個別フォロー（電話/メール） | 商談化率15%→25%に改善。必要リード数が4割減 | 人件費のみ |
| 助成金ガイドPDFの無料ダウンロード設置 | LP訪問者の追加CVポイント。月5-10件のリード追加見込み | 初回制作費のみ |
| 介護業界団体・社労士事務所からの紹介チャネル構築 | 広告費ゼロの高品質リード。月3-5件見込み | 営業工数 |
| ウェビナーを月3回に増やす | リード母数の拡大。月10-15件の追加見込み | Zoom費用のみ |

**特にウェビナー参加者への個別フォロー強化は必須。** これにより商談化率が改善すれば、リード数が少なくても商談数を確保できる。

### プラン選択の判断基準

- **プランA（月30万円）**: 初期は月1件の成約で粗利32万円程度。LP・コンテンツが自社対応のため固定費が低く、赤字リスクが小さい。リード数が22-33件と少なく、ファネルのどこかが下振れすると成約ゼロの月が出る可能性がある。最小限の投資で市場の反応を検証したい場合に適する。
- **プランB（月75万円）**: 初期1-3ヶ月は利益が薄いが、4ヶ月目以降は月4-6件の成約で粗利173-297万円。LP・コンテンツ制作費がかからない分、広告費に集中できる。**立ち上げ期の推奨プラン。初月は検証フェーズとして30万円で開始し、LP CVRの実数値を確認してから75万円に引き上げる段階的投入を推奨する（詳細は「予算と流入数の整合性評価」参照）。**
- **プランC（月105万円）**: リード数71-97件で最も安定するが、初期の投資額も大きい。安定期に入ってからプランBから移行する拡大プランとして位置づけるのが合理的。

## 2. ターゲットペルソナ

### ペルソナA：中小規模介護事業所の経営者・施設長
- **属性**: 40-60代、1-3拠点運営、従業員10-50名
- **課題**: 慢性的な人手不足、記録業務の負担、ICT化の遅れ
- **心理**: 「AI」という言葉に漠然とした興味はあるが、何から始めればよいかわからない
- **情報収集**: 介護業界メディア、介護展示会、同業者の口コミ、Facebook

### 4-11. 郵便DM（高優先）

介護事業所は日常業務がオフライン中心であり、施設長・管理者は現場にいることが多い。郵便DMは「手元に残る」「回覧される」という特性があり、デジタル広告が届きにくい層への有力なアプローチ手段となる。

#### 送付対象リストの構築

| リストソース | 特徴 |
|------------|------|
| WAM NET（福祉医療機構）の事業所検索 | 全国の介護事業所が網羅、施設種別で絞込み可能 |
| 介護サービス情報公表システム | 厚生労働省管轄、事業所名・所在地・管理者名が公開 |
| 法人リスト購入（帝国データバンク等） | 業種コード「介護」で抽出、決裁者名付き |
| 展示会・ウェビナー参加者リスト | 接点のある見込み客への追撃DM |

- **初回送付規模**: 1,000〜3,000通（地域またはエリアを絞ってテスト）
- **ターゲット絞込み**: 従業員10名以上の法人（助成金活用メリットが大きい層）

#### DM構成案

**封筒（A4サイズ）の中身:**

1. **挨拶状（A4 1枚）**
   - 施設長・管理者宛の名前入り
   - 「介護現場のAI活用で年間○○時間の業務削減」という問題提起
   - ウェビナーへの招待

2. **介護校パンフレット（A4 三つ折りまたは4P）**
   - 介護現場のAI活用事例（記録業務、ケアプラン、シフト等）
   - カリキュラム概要
   - No.1実績・堀江貴文プロデュースの信頼性
   - 受講者の声

3. **助成金ガイドチラシ（A4 1枚）**
   - 「受講料の最大75%が助成金対象」を大きく表示
   - 人数別の実質負担額シミュレーション表（料金表PDFの内容を活用）
   - 助成金の対象条件フローチャート

4. **ウェビナー招待チラシ（A5 1枚）**
   - 次回ウェビナーの日時・テーマ
   - QRコード → ウェビナー申込みページ（UTM付き）
   - 「参加無料」を強調

5. **返信用ハガキまたはFAX用紙**
   - 資料請求 / ウェビナー申込み / 無料相談希望をチェック式で
   - FAX返信は介護業界ではまだ有効な手段

#### DM送付パターン

| パターン | タイミング | 目的 |
|---------|----------|------|
| 新規開拓DM | 月1回（リスト入替） | 新規リード獲得 |
| ウェビナー告知DM | ウェビナー2週間前 | 申込み促進 |
| フォローDM | Web会議後1週間 | 検討中の法人への後押し |
| 季節DM | 4月（新年度予算）/ 10月（下期予算） | 予算確保のタイミングに合わせた訴求 |

#### 効果測定

- QRコードにUTMパラメータを付与し、DMからのLP流入を計測
- 返信ハガキ/FAXの返送率をトラッキング
- DM送付リストとCRM上の商談を紐づけて成約率を計測
- **目標反応率**: 0.5〜1.5%（DM業界平均は0.5〜1.0%、助成金訴求で上振れを期待）

#### コスト試算（1,000通送付の場合）

| 項目 | 単価 | 合計 |
|------|------|------|
| 印刷（封筒+中身4点） | 約80円/通 | 80,000円 |
| 郵送料（定形外） | 約120円/通 | 120,000円 |
| リスト購入/整備 | - | 30,000円 |
| デザイン制作（初回） | - | 50,000円 |
| **合計** | | **約28万円/回** |

※2回目以降はデザイン費不要で約23万円/回。反応率1%で10件のリード獲得の場合、リード単価は約2.3〜2.8万円。法人契約の単価を考えれば十分に投資対効果が見込める。

#### デジタル施策との連動

```
DM到着 → QRコードでLP訪問 → ウェビナー申込み or 資料請求
  ↓                              ↓
FAX/ハガキ返信          メールシーケンス開始
  ↓                              ↓
電話フォロー              ウェビナー参加
  ↓                              ↓
Web会議予約      ←←←←      Web会議予約
  ↓
法人契約
```

DMは単独施策ではなく、デジタル施策への入口として機能させる。特にDM到着後1週間以内にFacebookリターゲティング広告を重ねると、「紙でもWebでも見た」という複数接点効果で信頼感が高まる。

### 予算配分の方針

- **デジタル広告（Facebook + Google）に予算の60%以上を集中** - CPAが計測可能で、効果の悪い施策を即座に停止・改善できる
- **DMは全体の13-15%に抑制** - 送付数を絞り、デジタルが届きにくい層への補助チャネルとして運用
- **DMの送付数は少量から開始し、反応率が実証されてから拡大する** - 初期は500通テストで反応率を検証し、効果が確認できた場合のみ増やす

### チャネル別予算の動的運用方針

上記の予算配分はあくまで初期設定であり、固定ではない。介護業界は意思決定者の年齢層が高く、デジタル広告のリーチが想定通りに機能するかは未検証である。以下のプロセスで、データに基づいた予算再配分を行う。

#### ステップ1: 並行テスト（1-2ヶ月目）
- デジタル広告（Facebook・Google）を主軸で出稿開始
- 同時期にDM 500通のテスト送付を実施
- 両チャネルのCPA（リード獲得単価）を計測・比較

#### ステップ2: CPA比較に基づく判断（3ヶ月目）

| 結果パターン | 判断 |
|------------|------|
| デジタル広告のCPAが良好、DMの反応率が低い | 現行配分を維持。DMは最小限に縮小 |
| デジタル広告のCPAが想定より高い、DMの反応率が良好 | **DM予算を20-30%まで引き上げ、デジタル広告を縮小** |
| 両方とも良好 | 総予算を増額し、両チャネルを拡大 |
| 両方とも不調 | クリエイティブ・ターゲティングの見直しを優先 |

#### ステップ3: 四半期ごとの再評価
- 3ヶ月ごとにチャネル別CPAとファネル転換率を比較レビュー
- 効果の高いチャネルに予算を傾斜配分
- 効果が出ないチャネルは停止も含めて判断

**重要: 「デジタルが正解」という前提を持たない。** 介護業界の特殊性（現場中心の業務、高い平均年齢、FAX・郵便が現役のメディア）を考慮し、データで効果が実証されたチャネルに予算を集中させる柔軟な運用を行う。

## 10. 補足：助成金訴求を活かした無料ツール施策

### 助成金シミュレーター（リード獲得ツール）

LP内に設置する簡易計算ツール：
- 入力: 受講人数、企業規模（中小/大企業）
- 出力: 受講料総額、助成金額、実質負担額、月額換算
- CTA: 「詳しい助成金の相談はこちら」→ Web会議予約

このツールは資料請求やウェビナー申込みと並ぶ第3のCVポイントとして機能し、助成金に興味のある高意図リードを獲得できる。
