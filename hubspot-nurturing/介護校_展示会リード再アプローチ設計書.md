# 展示会獲得リード 再アプローチ＋ナーチャリング設計書
# ホリエモンAI学校 介護校

**対象:** 展示会・セミナー等で名刺交換・接点を持ったが、その後フォローが途絶えているリード
**目的:** 休眠リードの再活性化 → 商談化・受講申込み
**ツール:** HubSpot（ワークフロー・メール配信）

---

## 設計の考え方

展示会リードは資料請求リードと異なり、以下の特性があります。

| 特性 | 対策 |
|---|---|
| 一度face-to-faceで接点があった | 「ご挨拶した〇〇です」と関係性を想起させる |
| 時間が経過している（記憶が薄い） | 押しつけず、まず価値提供から入る |
| 検討意欲が不明 | 返信・クリックなどの反応を見て温度感を判断 |
| 名刺情報のみ（メール未配信の場合も） | 最初のメールは短く・読みやすく |

---

## シーケンス全体図

```
【配信開始】手動またはリスト一括エンロール
  ↓ Day 0
【Mail 1】再接触・ご挨拶メール（短文・圧迫感なし）
  ↓ 3日後
【Mail 2】介護現場の最新AI活用トレンド（価値提供）
  ↓ 7日後
【Mail 3】「こんな課題ありませんか？」共感メール
  ↓ 12日後
【Mail 4】助成金で実質70,000円/年になる話
  ↓ 17日後
【Mail 5】事例紹介（Before/After）
  ↓ 23日後
【Mail 6】無料相談 or ウェビナーへの招待
  ↓ 30日後
【Mail 7】最終アプローチ（反応なしの場合のみ送信）
```

## HubSpot 自動化設定

### エンロール条件（2パターン）

**パターン①：一括エンロール（既存の展示会リストを手動でインポート）**
- コンタクトリストを作成（展示会獲得・未アクティブ）
- ワークフローで「リスト登録」をトリガーに設定

**パターン②：自動エンロール（今後の展示会リード用）**
- 「リードソース = 展示会」のコンタクトが追加されたとき
- かつ「最終メール送信日 = 未設定 or 90日以上前」

---

### 送信スケジュール

| ステップ | タイミング | 送信時間 |
|---|---|---|
| Mail 1 | Day 0（エンロール日） | 10:00 |
| Mail 2 | Day 3 | 10:00 |
| Mail 3 | Day 7 | 10:00 |
| Mail 4 | Day 12 | 10:00 |
| Mail 5 | Day 17 | 10:00 |
| Mail 6 | Day 23 | 10:00 |
| Mail 7 | Day 30（反応なし条件付き） | 10:00 |

---

### 反応に応じた分岐（HubSpotのIF/THENブランチ）

```
Mail 3送信後
  ├─ リンククリックあり
  │    → Mail 4を翌日に前倒し送信
  │    → 担当者に「反応あり通知」を送る
  └─ 反応なし
       → 通常スケジュール継続

Mail 6送信後
  ├─ ウェビナー申込み or 相談申込み
  │    → シーケンス終了・商談管理に移行
  └─ 反応なし
       → 7日後にMail 7を送信
            └─ Mail 7後も反応なし
                 → 「休眠リスト」に移動
                 → 3ヶ月後に単発メール1通（季節の業界情報）
```

---

### 離脱条件（ゴール）

- 個別相談フォーム送信
- ウェビナー申込みフォーム送信
- ライフサイクルステージ = SQL または Opportunity
- 配信停止

---

### 資料請求シーケンスとの違い

| 項目 | 資料請求シーケンス | 展示会リード再アプローチ |
|---|---|---|
| 第1通目のトーン | 「資料をお届け」感謝 | 「ご挨拶した〇〇です」関係性想起 |
| 配信間隔 | 2〜3日間隔（短め） | 3〜7日間隔（ゆっくり） |
| 内容の入り方 | サービス説明から入る | トレンド・価値提供から入る |
| 最終メール | 期間限定の緊迫感 | 「最後のご連絡」＋配信停止を提示 |
| 目的 | 温度感を上げて商談化 | まず関係を再構築してから商談化 |

## 今後追加が必要な情報

| 不足情報 | 用途 |

| 担当者名・署名 | 全メールの差出人設定 |
| 展示会名・開催時期 | Mail 1の「先日の〇〇展示会で」に記載 |
| ウェビナー開催日程・URL | Mail 6のリンク先 |

| 既存展示会リストのCSVデータ | 一括インポート用 |
