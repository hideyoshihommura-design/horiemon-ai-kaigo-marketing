# 展示会獲得リード ウェビナー活用ナーチャリング設計書
# ホリエモンAI学校 介護校

**対象:** 展示会・セミナー等で名刺交換したが、連絡が途絶えている休眠リード
**目的:** ウェビナーを起点に関係を再構築 → 商談化・受講申込み
**ウェビナー:** 4月9日（水）13:30〜14:00 / AIが写真1枚で対応を指示してくれる時代へ

---

## 設計の考え方

連絡が空いている展示会リードへのアプローチは「いきなり売り込まない」が鉄則。
まず関係性を再構築し、価値あるコンテンツ（ウェビナー）を入り口にすることで
「押しつけられた」ではなく「役に立ちそう」と感じてもらう流れを作る。

```
再接触（関係想起）
  ↓
価値提供（トレンド情報）
  ↓
ウェビナーへの誘導（具体的テーマで興味喚起）
  ↓
ウェビナー参加
  ↓
参加者：御礼 → 助成金 → 個別相談
未参加者：録画案内 → 課題共感 → 個別相談
```

## HubSpot 自動化設定

### エンロール条件

**既存展示会リスト（一括）：**
- コンタクトリスト「展示会獲得リード」をHubSpotにインポート
- ワークフローで「リスト登録」をトリガーに設定

**今後の展示会リード（自動）：**
- リードソース = 展示会 のコンタクトが追加されたとき
- かつ 最終メール送信日 = 未設定 or 90日以上前
