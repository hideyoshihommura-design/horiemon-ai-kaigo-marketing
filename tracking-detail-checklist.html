タグが本当に発火しているかは、これらのツールで「目で見て」確認します。設定変更後・本番反映後に必ずチェックしてください。

GA4 リアルタイムレポート

GA4 公式

直近30分間のアクセス・イベントをリアルタイムで表示。自分でLPを開いて、ちゃんとイベントが届くか確認

analytics.google.com → レポート → リアルタイム

GA4 DebugView

GA4 公式

自分のブラウザだけのデバッグ環境。GA Debugger拡張機能と組合せで詳細イベント解析

管理 → DebugView

Google Tag Assistant

Chrome拡張

ページにあるGoogleタグ（GA4・Google Ads・GTM）を全部検出して発火状況を表示

tagassistant.google.com

Meta Pixel Helper

Chrome拡張

MetaのPixelをページから検出。発火イベント・パラメータが一覧表示

Chromeウェブストアで「Meta Pixel Helper」検索

Meta Events Manager テスト

Meta 公式

テストブラウザIDを発行し、特定ブラウザのイベントだけリアルタイム監視

Events Manager → テストイベント

LINE Tagダッシュボード

LINE 公式

送信履歴・テストモードでタグ動作確認

admanager.line.biz → 共有ライブラリ

GTMプレビューモード

GTM 公式

公開前の変更がどう動くか自ブラウザでテスト。タグ発火順序・データレイヤー値を確認

tagmanager.google.com → プレビュー

Chrome DevTools Network

ブラウザ内蔵

F12 → Network → 「fbevents」「collect」「google-analytics」等で検索し送信内容を直接確認

どのブラウザでも F12
