STEP 0.5フォント・文字サイズの設定

読みやすさとメリハリを出すために、フォントと文字サイズを統一します。

推奨フォント

優先度フォント名特徴
第1候補Noto Sans JPGoogle提供。読みやすく、モダンな印象。Web表示に最適
第2候補ヒラギノ角ゴシックMac/iPhone標準。高級感があり、プロフェッショナルな印象
第3候補メイリオ / 游ゴシックWindows標準。幅広い環境で表示可能
ペライチでの設定: 「ページ設定」→「デザイン設定」→「フォント」から選択。「ゴシック体」を選ぶと Noto Sans JP が適用されます。

文字サイズのメリハリ（重要）

文字の大きさに差をつけることで、読者の視線を誘導し、重要な情報を強調します。

要素PC表示スマホ表示太さ用途
メインコピー40〜48px28〜32px極太（900/Black）ファーストビューの見出し
セクション見出し28〜32px22〜26px太字（700/Bold）各セクションのタイトル
サブ見出し20〜24px18〜20px太字（700/Bold）セクション内の小見出し
本文16〜18px15〜16px標準（400/Regular）説明文、リスト
補足・注釈12〜14px11〜13px標準（400/Regular）※印の注意書きなど
数字（強調）36〜48px28〜36px極太（900/Black）「95%」「240+」など
ラベル・バッジ12〜14px11〜13px太字（700/Bold）「助成金で75%OFF」など
CTAボタン16〜18px15〜17px太字（700/Bold）ボタン内のテキスト

メリハリをつけるポイント

  1. サイズ差は1.5倍以上：メインコピー（48px）と本文（16px）の差は3倍。差が小さいと単調に見える
  2. 太さを変える：見出しは「太字」or「極太」、本文は「標準」。同じサイズでも太さで差がつく
  3. 数字は大きく：「95%」「240+」「5,833円」など、数字はインパクトを出すため大きめに
  4. 余白も重要：文字の周りに余白を取ると、より大きく・読みやすく見える

ペライチでの文字サイズ変更方法

  1. 変更したいテキストをクリック
  2. テキスト編集画面が開く
  3. 「文字サイズ」のプルダウンから選択（S / M / L / XL など）
  4. 「太字」ボタン（B）で太さを変更
  5. プレビューで確認し、スマホ表示もチェック
⚠️ スマホ表示の注意
・PC用に大きくしすぎると、スマホで文字が画面をはみ出すことがある
・必ずスマホプレビューで確認する（介護現場はスマホ閲覧が約60%）
・ペライチは自動でレスポンシブ対応するが、極端に大きい文字は手動調整が必要な場合も

ページ内リンクの設定方法

  1. リンク先にしたいセクション（例: ウェビナー申込み）の「編集」を開く
  2. 「セクションID」に webinar と入力して保存
  3. ファーストビューのボタンの「リンク設定」を開く
  4. リンク先URLに #webinar と入力
  5. 資料ダウンロードも同様（セクションID: download、ボタンリンク: #download）

背景画像について

  • 現在の堀江貴文の写真入り画像は維持してOK
  • テキストが読みにくい場合 → セクション編集 →「背景設定」→「オーバーレイ」→ 黒を選択 → 不透明度 30-40%

Q&A（以下をそのままコピペ）

Q1: AIの知識がまったくないスタッフでも受講できますか？
はい。受講者の95%がAI未経験者です。PC操作ができる程度のスキルがあれば安心して受講いただけます。
Q2: 助成金は本当に使えますか？手続きは難しくないですか？
雇用保険に加入している社員が業務時間内に受講する場合、人材開発支援助成金の対象になります。提携社労士が申請手続きを無料でサポートしますので、手続きの負担はほとんどありません。
Q3: 1名だけでも受講できますか？
法人プランは2名以上からお申込みいただけます。1名での受講をご希望の場合は「個人プラン」をご利用ください。（詳細は下部に記載）
Q4: 受講期間はどのくらいですか？
1年間の受講期間です。カリキュラムは2ヶ月〜6ヶ月で設計し、残りの期間は実際の業務改善に伴走型でサポートします。
Q5: どんなAIツールを学べますか？
ChatGPT、Gemini、Copilotなど主要な生成AIツールに対応しています。社内環境に合わせて基礎学習の内容をカスタマイズします。
Q6: 忙しくて受講の時間が取れるか心配です。
動画講義は好きな時間に視聴可能です。助成金の要件として全員で合計10時間の視聴が必要ですが、10名受講なら1名あたり1時間で要件クリアです。
