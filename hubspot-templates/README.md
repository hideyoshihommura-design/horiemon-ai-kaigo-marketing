# HubSpot デザインマネージャー テンプレート設置手順

## ファイル

- `webinar-lp.html` — ウェビナー専用ランディングページのテンプレート

## 設置手順

### 1. デザインマネージャーを開く

HubSpotにログイン → **マーケティング → ファイルとテンプレート → デザインツール**

### 2. フォルダを作成

左パネルで「ファイル」→ 右クリック →「新しいフォルダ」→ フォルダ名を `kaigo-templates` に設定

### 3. テンプレートファイルを作成

1. 作成した `kaigo-templates` フォルダを選択
2. 左上の「+ 新規ファイル」→「HTMLとHubL」を選択
3. ファイル名を `webinar-lp.html` に設定
4. **`webinar-lp.html` の中身をすべてコピーして貼り付け**
5. 右上の「公開」をクリック

### 4. テンプレートとして認識されたか確認

右パネルに以下が表示されていれば成功：
- テンプレートタイプ: ページ
- ラベル: ウェビナー専用LP - 介護校

### 5. ランディングページで使用する

1. **マーケティング → ランディングページ → 「作成」**
2. テンプレート選択画面で「ウェビナー専用LP - 介護校」を選択
3. ページ名を入力して「ページを作成」

### 6. コンテンツを編集

テンプレート内の各モジュール（`{% module %}` で定義された部分）は、
ページエディタの左パネルからクリックして編集できます：

| モジュール | 編集内容 |
|-----------|---------|
| ヒーロータイトル | ウェビナーのタイトル・サブタイトル |
| 開催日 | 開催日時の表示テキスト |
| お悩みリスト | ターゲットの課題（追加・変更可） |
| メリット1〜3 | ウェビナーで得られること |
| プログラム内容 | 当日のプログラム |
| 講師情報 | 講師名・プロフィール |
| 講師写真 | 講師の写真画像 |
| 開催概要 | 日時・会場・定員など |
| ウェビナー申込みフォーム | HubSpotフォームを選択 |
| FAQ | よくある質問の追加・変更 |

### 7. フォームを紐づける

1. ページエディタで「ウェビナー申込みフォーム」モジュールをクリック
2. 左パネルで「フォーム」→ 事前に作成した「ウェビナー申込みフォーム」を選択
3. 送信後メッセージが表示されることを確認

### 8. ページ設定

1. 上部の「設定」タブをクリック
2. **ページタイトル**: 「介護現場のAI活用最前線 無料ウェビナー｜ホリエモンAI学校 介護校」
3. **メタディスクリプション**: 「介護事業所の経営者・施設長向け無料ウェビナー。AIで記録業務を50%削減する方法と、助成金で研修費75%OFFにする手順を解説。」
4. **URL スラッグ**: `webinar-kaigo-ai`

### 9. プレビュー・公開

1. 右上の「プレビュー」でPC・スマホ表示を確認
2. 問題なければ「公開」をクリック

## 注意事項

- テンプレート内の `{% module %}` タグはHubLの記法です。デザインマネージャーで正しく認識されます
- スタイル（CSS）はテンプレート内に含まれているため、別ファイルの設置は不要です
- フォームモジュールは、ページ作成後にエディタ上で既存のHubSpotフォームを選択する必要があります
- ウェビナーの日程が変わるたびにページを複製して新しいページを作ることを推奨します
