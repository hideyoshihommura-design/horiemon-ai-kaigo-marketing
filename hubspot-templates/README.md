# HubSpot デザインマネージャー テンプレート設置手順

## ファイル

| ファイル名 | 用途 | 広告誘導先 |
|-----------|------|----------|
| `webinar-lp.html` | ウェビナー専用LP | Facebook・Google広告 |
| `document-request-lp.html` | 資料請求専用LP | Facebook・Google・LINE広告 |

### 3ページ体制での役割分担

```
┌─────────────────────────────────────────────────────────────┐
│  メインLP（ペライチ）                                        │
│  - 総合情報・SEO・指名検索                                   │
│  - Google検索、SNSからの流入                                 │
└─────────────────────────────────────────────────────────────┘
         ↑ 詳しく知りたい人
         │
┌────────┴────────┬─────────────────────────────────────────────┐
│                  │                                             │
│  ウェビナー専用LP │  資料請求専用LP                            │
│  (HubSpot)       │  (HubSpot)                                 │
│  - CTA: 申込み    │  - CTA: 資料DL                             │
│  - Facebook広告   │  - Facebook・Google・LINE広告              │
│  - Google広告    │  - リターゲティング広告                      │
└──────────────────┴─────────────────────────────────────────────┘
```

## 設置手順

### 1. デザインマネージャーを開く

HubSpotにログイン → **マーケティング → ファイルとテンプレート → デザインツール**

### 2. フォルダを作成

左パネルで「ファイル」→ 右クリック →「新しいフォルダ」→ フォルダ名を `kaigo-templates` に設定

### 3. テンプレートファイルを作成

1. 作成した `kaigo-templates` フォルダを選択
2. 左上の「+ 新規ファイル」→「HTMLとHubL」を選択
3. ファイル名を `webinar-lp.html` に設定
4. **`webinar-lp.html` の中身をすべてコピーして貼り付け**
5. 右上の「公開」をクリック

### 4. テンプレートとして認識されたか確認

右パネルに以下が表示されていれば成功：
- テンプレートタイプ: ページ
- ラベル: ウェビナー専用LP - 介護校

### 5. ランディングページで使用する

1. **マーケティング → ランディングページ → 「作成」**
2. テンプレート選択画面で「ウェビナー専用LP - 介護校」を選択
3. ページ名を入力して「ページを作成」

### 6. コンテンツを編集

テンプレート内の各モジュール（`{% module %}` で定義された部分）は、
ページエディタの左パネルからクリックして編集できます：

| モジュール | 編集内容 |
|-----------|---------|
| ヒーロータイトル | ウェビナーのタイトル・サブタイトル |
| 開催日 | 開催日時の表示テキスト |
| お悩みリスト | ターゲットの課題（追加・変更可） |
| メリット1〜3 | ウェビナーで得られること |
| プログラム内容 | 当日のプログラム |
| 講師情報 | 講師名・プロフィール |
| 講師写真 | 講師の写真画像 |
| 開催概要 | 日時・会場・定員など |
| ウェビナー申込みフォーム | HubSpotフォームを選択 |
| FAQ | よくある質問の追加・変更 |

### 7. フォームを紐づける

1. ページエディタで「ウェビナー申込みフォーム」モジュールをクリック
2. 左パネルで「フォーム」→ 事前に作成した「ウェビナー申込みフォーム」を選択
3. 送信後メッセージが表示されることを確認

### 8. ページ設定

1. 上部の「設定」タブをクリック
2. **ページタイトル**: 「介護現場のAI活用最前線 無料ウェビナー｜ホリエモンAI学校 介護校」
3. **メタディスクリプション**: 「介護事業所の経営者・施設長向け無料ウェビナー。AIで記録業務を50%削減する方法と、助成金で研修費75%OFFにする手順を解説。」
4. **URL スラッグ**: `webinar-kaigo-ai`

### 9. プレビュー・公開

1. 右上の「プレビュー」でPC・スマホ表示を確認
2. 問題なければ「公開」をクリック

## 注意事項

- テンプレート内の `{% module %}` タグはHubLの記法です。デザインマネージャーで正しく認識されます
- スタイル（CSS）はテンプレート内に含まれているため、別ファイルの設置は不要です
- フォームモジュールは、ページ作成後にエディタ上で既存のHubSpotフォームを選択する必要があります
- ウェビナーの日程が変わるたびにページを複製して新しいページを作ることを推奨します

## 資料請求LP (`document-request-lp.html`) 設置手順

### 1. テンプレートファイルを作成

1. デザインマネージャーで `kaigo-templates` フォルダを選択
2. 「+ 新規ファイル」→「HTMLとHubL」を選択
3. ファイル名を `document-request-lp.html` に設定
4. **`document-request-lp.html` の中身をすべてコピーして貼り付け**
5. 右上の「公開」をクリック

### 2. テンプレートとして認識されたか確認

右パネルに以下が表示されていれば成功：
- テンプレートタイプ: ページ
- ラベル: 資料請求専用LP - 介護校

### 3. ランディングページで使用する

1. **マーケティング → ランディングページ → 「作成」**
2. テンプレート選択画面で「資料請求専用LP - 介護校」を選択
3. ページ名を入力して「ページを作成」

### 4. コンテンツを編集

| モジュール | 編集内容 |
|-----------|---------|
| ヒーロータイトル | メインコピー（価格訴求含む） |
| 資料タイトル | 資料のタイトル表示 |
| 資料でわかること | 3つのベネフィット |
| No.1バッジ | 4冠バッジの表示 |
| 資料請求フォーム | HubSpotフォームを選択 |
| 料金訴求 | 助成金による価格訴求 |
| 4つの特徴 | サービスの特徴 |
| FAQ | よくある質問 |

### 5. フォームを紐づける

**重要**: 資料請求フォームは項目を最小限に（推奨：2項目）

1. 事前にHubSpotで「資料請求フォーム」を作成
   - **メールアドレス**（必須）
   - **法人名 / 施設名**（必須）
   - ※電話番号・役職は任意でも可
2. ページエディタで「資料請求フォーム」モジュールをクリック
3. 左パネルで作成したフォームを選択

### 6. ページ設定

| 項目 | 推奨設定 |
|------|---------|
| ページタイトル | 「介護AI研修の資料請求｜助成金で75%OFF｜ホリエモンAI学校」 |
| メタディスクリプション | 「介護事業所向けAI研修の詳細資料を無料ダウンロード。助成金活用で実質月5,833円/人。240講座以上の動画講義とチャットサポートで、AI未経験でも安心。」 |
| URL スラッグ | `document-kaigo-ai` |

### 7. 広告との連携

#### Facebook広告
- キャンペーン目的: リード獲得 または コンバージョン
- 誘導先: `https://yourdomain.com/document-kaigo-ai`
- ターゲティング: 介護施設管理者・経営者 40-65歳

#### Google広告
- キャンペーン: 検索広告（「介護 AI研修 資料」等）
- コンバージョン: フォーム送信をコンバージョンに設定

#### LINE広告
- 配信先: LINE NEWS・タイムライン・VOOM
- 目的: 資料ダウンロードCV

### 8. A/Bテスト（推奨）

HubSpotの「ページの複製」機能で以下をテスト：
- ヒーロータイトルのコピー（課題訴求 vs 価格訴求）
- CTAボタンの文言（「無料でダウンロード」vs「今すぐ資料を見る」）
- フォーム項目数（2項目 vs 3項目）

## 2つのLPの使い分け

| 項目 | ウェビナーLP | 資料請求LP |
|------|-------------|-----------|
| 目的 | ウェビナー申込み | 資料ダウンロード |
| フォーム項目 | 4-5項目 | 2項目（最小限） |
| CVまでの心理的ハードル | 中（日時を確保する必要） | 低（すぐ読める） |
| リードの温度感 | 高（能動的に参加） | 中（まず情報収集） |
| フォロー優先度 | 高 | 中 |
| 適した広告 | 認知〜検討層 | 情報収集層・リターゲ |

### 推奨フロー

```
[広告] → [資料請求LP] → 資料DL → メールシーケンス → ウェビナー誘導 → 商談
           ↓
        リターゲ広告
           ↓
       [ウェビナーLP] → 申込み → 参加 → 商談
```
